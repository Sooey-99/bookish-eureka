Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The paragraph element contains information to be shown as regular text content.

CSS

  • CSS is used to add the "meat" or the design to the bare structure of HTML.
  • CSS is always added after your HTML founndations are set.
  • After this you'll look to add Javascript.

GIT

  • Git is key for version control.
  • Use Git-hub to save your work in a repository or repo for short.
  • In git you'll also be able to colaborate with others on projects simultaiously using different branches.

Javascript

  • Javasccript will give your website or app functionality and take your project from a standard 2D page, to a 3D interactive page.
